### Status
#### `error`
Uttrykker at noe mangler eller er skrevet feil.
<Canvas of={FhiTextInputStories.Error} />

### Form Controls
#### `readonly`
Gjør at feltet ikke kan endres. Feltet vil fortsatt kunne fokuseres og vil automatisk bli inkludert i FormData.
<Canvas of={FhiTextInputStories.Readonly} />
#### `disabled`
Gjør at feltet ikke kan endres, fokuseres eller automatisk inkluderes i FormData. Unngås hvis man kan.
<Canvas of={FhiTextInputStories.Disabled} />
